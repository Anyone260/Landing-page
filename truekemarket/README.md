# TruekeMarket

## Sobre el proyecto

TruekeMarket es un aplicativo dirigido a estudiantes universitarios. La idea es que los universitarios puedan hacer trueques de forma facil y sencilla. Lo que hace diferente a TruekeMarket es que todos los usuarios son estudiantes registrados con su ID universitario, lo que da mayor seguridad y confianza al momento de intercambiar.

## A quien va dirigido

Estudiantes universitarios que quieran intercambiar cosas entre ellos de manera segura.

## Que ofrece

- Registro con ID universitario
- Trueques entre estudiantes verificados
- Facil de usar
- Seguridad porque todos estan registrados
